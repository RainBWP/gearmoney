# DAM
## Objetivo
Aplicar
- Formularios
- Validación
- Navegación
- Manejo de Estado

## Instrucciones
Crear una app que
### Pantalla 1 : Formulario
Debe capturar
1. Nombre del Alumno
2. Matrícula
3. Carrera
#### Reglas de validación
- Ningún campo puede estar vacío
- La matrícula debe ser Numerica
#### Boton Registrar
Debe:
1. Validar Formulario
2. Guardar Datos
3. Navegar a segunda pantalla
### Pantalla 2 : Resultado
Debe Mostrar:
- Alumno Registrado
    - Nombre
    - Matrícula
    - Carrera

## Restricciones
Solo pueden usar:
- StatefulWidget
- StatelessWidget
- setState
- Text
- TextFormField
- ElevatedButton
- Navigator
- Scaffold
- Form
- GlobalKey
